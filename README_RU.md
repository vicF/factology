# Фактология

**Гибкая система для описания и визуализации связей между любыми сущностями.**

Проект позволяет моделировать знания, события, понятия, объекты и их взаимоотношения в виде направленного графа. Каждая связь может иметь свой тип, а сущности – произвольные атрибуты. Система спроектирована для работы в браузере (серверное приложение) с планами на мобильные платформы.

## Основные возможности

- **Любые типы объектов**: люди, места, события, абстрактные понятия, файлы – всё, что угодно.
- **Произвольные типы связей** – определяйте свои отношения (например, «включает», «является родителем», «производит», «вызывает»).
- **Направленные связи** – стрелка показывает семантику отношения.
- **Группировка объектов** через бинарные связи (например, «элемент → принадлежит → множество»).
- **Обмен данными между приложениями** – экспорт/импорт данных в стандартных форматах (JSON, XML, gedcom в перспективе).
- **Серверная часть** написана на PHP. Предоставляет API для работы клиентских приложений.
- **Стандартный клиент** на Vue 3.

## Статус репозитория

> **Статус**: MVP (минимально жизнеспособный продукт) – базовая работа с сущностями и связями. Минимальная визуализация графа.

## Ближайшие планы

- Экспорт и импорт данных
- Поддержка широкого спектра дат
- Работа с изображениями
- Адаптация под мобильные приложения
